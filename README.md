# A1 手语识别方向 Demo

这是一个基于 MediaPipe Hands 的浏览器端手部关键点检测 Demo。

## 功能

- 摄像头实时检测手部 21 个关键点
- Canvas 可视化手部骨架和关键点
- 统计检测到的手数量
- 区分左手和右手
- 简单识别伸出手指数，并映射为握拳、数字 1-5、张开手掌

## 本地运行

直接用浏览器打开 `index.html`，允许摄像头权限即可。

如果浏览器限制本地摄像头访问，可以在当前目录启动一个静态服务器：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## GitHub Pages 部署

1. 新建 GitHub 仓库，把本目录的文件上传。
2. 在仓库设置中打开 Pages。
3. Source 选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/root`。
4. 保存后等待生成访问链接。
